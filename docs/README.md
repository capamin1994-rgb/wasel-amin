# منصة واصل - GitHub Pages

هذا المجلد يحتوي على صفحة الهبوط الثابتة لمنصة واصل الخاصة بـ GitHub Pages.

## كيفية النشر على GitHub Pages

### الخطوة 1: تفعيل GitHub Pages

1. اذهب إلى إعدادات المشروع على GitHub
2. انتقل إلى **Settings** → **Pages**
3. اختر **Deploy from a branch**
4. اختر:
   - **Branch**: `main` (أو الفرع الذي تستخدمه)
   - **Folder**: `docs` (هذا المجلد)
5. انقر على **Save**

### الخطوة 2: انتظر التفعيل

GitHub سيبني الصفحة تلقائياً وستكون متاحة على:
```
https://your-username.github.io/wasel
```

أو إذا كان المشروع تحت منظمة:
```
https://your-org.github.io/wasel
```

## ملاحظات

- الملف `index.html` هو نقطة الدخول الرئيسية
- جميع الأنماط CSS مدمجة في الملف (inline)
- الصفحة متجاوبة وتعمل على جميع الأجهزة
- تدعم المظهر الليلي (Dark Mode)

## تعديل المحتوى

يمكنك تعديل:
- النصوص والعناوين
- الروابط (غير الروابط الديناميكية)
- الألوان والأنماط في قسم `<style>`
- الصور والأيقونات
